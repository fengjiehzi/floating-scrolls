# 章节质量检查清单

## A. 硬性一致性：任一失败即退修

- [ ] 时间、地点、人物位置能从上一章连续推导
- [ ] 伤势、物品、资源和能力状态一致
- [ ] 没有人知道尚未获得的信息
- [ ] 世界规则、能力上限与代价没有被绕过
- [ ] 角色行为没有突破小说圣经中的底线，或突破已被充分铺垫并付出代价
- [ ] 新事实不与定稿正文或连续性台账冲突
- [ ] 人名、称谓、术语和叙事视角统一

## B. 章节有效性：任一失败原则上退修

- [ ] 一句话能说清本章核心任务
- [ ] 章初状态和章末状态不同
- [ ] 主角或视角角色作出了选择，而非只被事件拖着走
- [ ] 每场戏都有目标、阻力和结果
- [ ] 场景之间由“因此 / 但是”连接，而不是单纯“然后”
- [ ] 本章推进了主线、人物或关键信息中的至少两项
- [ ] 章末钩子来自本章因果，而非凭空制造意外

## C. 阅读质量：可局部修改

- [ ] 开头尽快进入具体人物、场景或问题
- [ ] 情绪主要由动作、选择、对话和感官细节呈现
- [ ] 对话人物可辨，且每段对话有意图
- [ ] 背景说明没有连续大段阻断当前行动
- [ ] 重复信息、无效往返和可删除段落已压缩
- [ ] 长短句与场景节奏匹配
- [ ] 没有滥用“突然、仿佛、不由得、微微、缓缓”等惯性词

## D. 定稿回写

- [ ] `审稿.md` 已标记“通过”
- [ ] 连续性台账只加入正文已确认的事实
- [ ] 故事总纲中的章节状态已改为“定稿”
- [ ] 项目状态已记录最近定稿章和下一步动作
- [ ] 若改变核心设定，变更记录已说明原因、影响和回查范围
