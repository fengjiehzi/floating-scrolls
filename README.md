# 万卷浮生 · Floating Scrolls

> AI 驱动的东方志怪典籍互动对战 Web 游戏

---

## 📖 项目简介

「万卷浮生」是一款融合古典名著阅读、AI 互动剧情、角色养成与跨作品 PVP 对战的 Web 游戏。玩家可导入古典名著（西游记、三国演义、水浒传、红楼梦、封神演义、白蛇传等），由 AI 生成互动剧情章节，在阅读中收集角色卡与法宝，最终让不同作品中的角色同台竞技——孙悟空对战迪迦、关羽对决哪吒，皆可实现。

### ✨ 核心特色

- 📖 **典籍阅读 + AI 剧情**：12 部预置名著，AI 根据原文生成分章节互动剧情，玩家选择影响故事走向
- 🎴 **24 个角色卡**：覆盖西游记、三国演义、水浒传、红楼梦、封神演义、白蛇传、山海经、史记、奥特曼 9 大来源
- 🏺 **31 件法宝**：金箍棒、青龙偃月刀、乾坤圈、风火轮等传奇道具
- ⚔️ **跨作品 PVP 对战**：WebSocket 实时对战，支持跨维度压制、等级压制、秒杀机制
- 🤖 **11 家国产大模型**：DeepSeek、智谱、Kimi、通义、MiniMax、豆包、阶跃、硅基流动、Mimo、OpenAI、Ollama
- 🎨 **玻璃拟态 UI**：动态背景、毛玻璃效果、流畅动画，移动端自适应

## 🚀 快速开始

### 环境要求

| 项目 | 要求 |
|------|------|
| Node.js | ≥ 18.0.0 |
| npm | ≥ 9.0.0 |
| 操作系统 | Windows / macOS / Linux |
| 浏览器 | Chrome / Edge / Firefox（最新版） |

### 安装与启动

```bash
# 1. 进入项目目录
cd 万卷浮生_Web应用

# 2. 安装依赖
npm install

# 3. 启动服务器
npm start
```

启动成功后访问：**http://localhost:8080**

### AI 配置（重要）

首次使用需在「🤖 AI 设置」页面配置 AI 服务商与 API Key，支持 11 家服务商的 OpenAI 兼容 + Anthropic 原生双协议。

## 📁 项目结构

```
万卷浮生/
├── 万卷浮生_Web应用/    # 主 Web 应用（Express + 原生 JS）
│   ├── server.js         # 服务器入口
│   ├── battle-engine.js  # 战斗引擎
│   ├── ws-server.js      # WebSocket 对战服务器

│   └── public/           # 前端静态资源
├── frontend/             # React 前端项目（Vite + TypeScript）
├── docs/                 # 技术文档与 ADR
└── README.md             # 项目说明
```

## 🎮 游戏玩法

| 模块 | 功能 |
|------|------|
| 📚 典籍库 | 浏览 12 部预置名著，支持上传自定义小说 |
| 📖 剧情主线 | AI 互动阅读，选择影响故事走向 |
| 🎴 角色卡库 | 24 个角色，按来源分组，S/A/B 三级 |
| 🏺 法宝阁 | 31 件法宝，按类型与稀有度分类 |
| ⚔️ 对战大厅 | 在线 PVP + 机器人对战，WebSocket 实时通信 |
| 🤖 AI 设置 | 11 家服务商配置，双协议支持 |
| 📁 存档管理 | 剧情进度、对战记录、排行榜 |

## 🛠️ 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Node.js + Express 4 |
| 数据库 | SQLite (better-sqlite3) |
| 实时通信 | WebSocket (ws) |
| 认证 | JWT + bcryptjs |
| AI 集成 | OpenAI 兼容 + Anthropic 原生双协议 |
| 前端 | 原生 HTML + JavaScript (ES6+) |
| React 前端 | Vite + TypeScript + Zustand |
| 样式 | 自定义 CSS（玻璃拟态） |
